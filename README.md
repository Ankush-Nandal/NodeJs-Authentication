## Installation

Repo link 
https://github.com/Ankush-Nandal/NodeJs-Authentication


> ***Install project with npm***       

```bash
  cd NodeJs-Authentication
  npm install
  inp install bcrypt
  npm install nodemon

```

***Start APP***

```bash 
npm start
```
***start using***
node index.js


    
## ***Features***

- User can create account 
- Password will store after Encryption in database
- Google Authentication
